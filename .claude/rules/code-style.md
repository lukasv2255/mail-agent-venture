# Styl kódu

- Preferuj funkcionální vzory před class-based
- Používej `const` místo `let` kde to jde
- Preferuj explicitní typy nad `any`
- Maximální délka funkce: ~50 řádků — pokud je delší, rozděl ji
- Názvy proměnných: popisné, anglicky
- Žádné magické čísla — použij pojmenované konstanty

## Formátování

- Odsazení: 2 mezery
- Uvozovky: single quotes (JS/TS), double (JSX atributy)
- Trailing comma: ano
- Semicolony: [ano / ne — vyber jedno]
